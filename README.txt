Demo för att visa hur man kan jobba mot Get a Newsletters API. 

XMLRPC biblioteket som används heter XMLRPC FOR PHP och finns att
ladda ner i sin helhet på http://sourceforge.net/projects/phpxmlrpc/

demo.php	Exempelfil för att visa hur klassen GAPI.class.php kan
		användas i PHP

GAPI.class.php  Klass som används för att koppla upp mot Get a Newsletter.

För att komma igång, prova att köra demo.php. 

Vill du läsa mer om hur APIt kan användas så besök
http://admin.getanewsletter.com/api/

Har du en fråga så skicka ett mejl till oss på
support@getanewsletter.com så hjälper vi dig.
